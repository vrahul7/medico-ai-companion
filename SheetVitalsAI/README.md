# SheetVitals AI 🩺📊

A standalone, high-performance clinical data analysis tool.

## Features
- **Zero Hallucination**: Deterministic data parsing with `pandas`.
- **Premium UI**: Modern, glassmorphic design for professional use.
- **AI Synthesis**: Intelligent clinical insights powered by Gemini 1.5 Flash.
- **Privacy First**: Local processing, no data persistence.

## How to Run
1. Open PowerShell in this directory.
2. Run `./run.ps1`.
3. Open `http://localhost:8002` in your browser.

## Tech Stack
- **Frontend**: Vanilla HTML/CSS/JS (High performance)
- **Backend**: Python FastAPI + Pandas
- **AI**: Google Gemini 1.5 Flash
